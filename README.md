# Pi-Top with ZeroROS

This is a project to run ZeroROS on a Pi-Top.

## Installation

Install with pip:

```bash
git clone https://github.com/miquelmassot/pitop_zeroros.git
cd pitop_zeroros
pip install -U --user -e .
```

## Tool to test the SLAMTEC RPLidar A1

```bash
cd pitop_zeroros
python tools/rplidar_test.py
```
